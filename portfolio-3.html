Qualifly - Crowdsourced Testing Platform

Description

  • Daniyal Aamir

  • qualifly.us

Qualifly is an on-demand testing automation service that is fast, easy, and reliable and is supported by a curated crowd of testers. It lowers the costs of testing for IT organizations (OPEX and CAPEX ) while delivering consistent, high-quality testing results. The platform offers Flexible models, easy and seamless engagement, faster testing cycles, and significantly reduces OPEX and CAPEX for the IT organizations.

  • Developed the front-end web page by using HTML5, CSS3, Bootstrap and JavaScript, taking advantage of the framework React.js.

  • Utilized MongoDB to perform various CRUD operations. Used mongoose for integration of MongoDB in the project.

  • Worked on Unit test by Test-Driven Development (TDD), writing test cases using Jest tests.
