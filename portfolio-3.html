Smart Brain

Description

  • Daniyal Aamir

  • smart-brain-frontend-app.herokuapp.com

Built a full web-based project where you can detect the face of a human image with the help of a Clarifai API. Built the frontend using React.js and backend using Express framework, Node.js and PostgreSQL database. Deployed all major components frontend, backend and database in Heroku.

  • Utilized React.js Eco-system in the development of the app for development efficiency

  • Developed a RESTful API using Express.js connected to a PostgreSQL database enabling users to register, sign-in and edit their profile information

* Note: This project is hosted via Heroku's free dyno plan, so it may take 30-60 seconds to load.
