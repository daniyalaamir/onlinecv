FEMA GO

Description

  • Daniyal Aamir

  • go.fema.gov

There are over 40 different FEMA grant programs. Each have their own distinct processes and web applications for determining and monitoring awards. GMM is an effort to streamline those processes for all funding opportunities and to provide a single web application, FEMA GO.

  • Implemented React/Redux UI application for U.S. FEMA to replace disconnected legacy systems in an AGILE environment.

  • Developed UI React.js components, Redux middleware, and full application views with federal 508 compliance regulations.

  • Created Jest unit tests.
